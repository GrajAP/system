# System

Minimal PHP entrypoint used as a quick redirect to the app login flow.

## Current behavior
- `index.php` redirects all traffic to `/pages/logowanie`.
- The project includes assets and basic page scaffolding under `pages/`.
- Contains a `TODO` placeholder for further development (currently intentionally lightweight).

## Goal
I used this as a fast scaffold for a simple server-side flow. It is intentionally small and currently ready for extension (auth, session layer, routing, and templates).

## Tech stack
- PHP
- Basic frontend assets (HTML/CSS/JS expected in `assets/` and `pages/`)

## Why this repo is kept
Simple project to show I work with backend basics in addition to frontend/mobile/desktop code, not only JS apps.
